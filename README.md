# Flutter - Angela Yu Course Projects

This repository contains sample Flutter projects completed as part of Angela Yu's Flutter course.

Projects
- dicee_flutter — Dice rolling app (UI & state)
- i_am_poor — Simple app demonstrating text and layout
- i_am_rich — Basic UI with centered content
- layout_challenge — Practicing layout widgets
- magic_8_ball — Simple logic + UI demo
- mi_card_flutter — Profile card UI
- xylophone_flutter — Sound playing app with buttons

Notes
- These projects were created with Flutter.
- Run `flutter pub get` inside a project's directory before running.
- The repo excludes build artifacts and IDE files via `.gitignore`.
